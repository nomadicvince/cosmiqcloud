About Me

Hi, I'm Vincent. I am a web developer specializing in web animation, WebGL, data visualization and full-stack web applications built with React, Node.js, WebAssembly or Rust. I am also an astronomical artist, travel journalist, photographer and 3D artist.

Web Development

I have been fascinated by web design and development since college. I feel the best way to create is to constantly explore cutting-edge concepts and push the envelope as much as possible.
